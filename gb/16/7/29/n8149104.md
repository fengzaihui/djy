<tr><td VALIGN=TOP width="626"><h2 align=center>墨尔本房价增长最快城区</h2>

<h6></h6>
<hr>
<p>【大纪元2016年07月29日讯】（大纪元记者张威廉<ahref="https://github.com/dahstt3965/djy/blob/master/gb/tag/%E5%A2%A8%E5%B0%94%E6%9C%AC.md#1">墨尔本</a>编译报导）在过去一年内，很多传统的蓝领阶层居住区受到青睐，这些地区的住宅中位价上涨幅度超过了20%，成为墨尔本<ahref="https://github.com/dahstt3965/djy/blob/master/gb/tag/%E6%88%BF%E4%BB%B7.md#1">房价</a>增长最快的地区。</p>
<p>据都门（Domain）新闻报导，位于<ahref="https://github.com/dahstt3965/djy/blob/master/gb/tag/%E5%A2%A8%E5%B0%94%E6%9C%AC.md#1">墨尔本</a>CBD东南方向27公里处的吉斯巴拉区（Keysborough）越来越受到买方的关注，因为这里的住宅价格明显低于滨海地带或距CBD中等距离的城区的住宅价格。在过去一年内，该区住宅价格中位数增长了22.5%，达到63.1万澳元。</p>
<p>都门集团的数据显示，高增长区的<ahref="https://github.com/dahstt3965/djy/blob/master/gb/tag/%E6%88%BF%E5%9C%B0%E4%BA%A7.md#1">房地产</a>中位数价格（根据今年前六个月的销售数据）及其涨幅如下：</p>
<ul>
<li>霍桑区（Hawthorn）：中位价211.65万澳元，增幅47.7%</li>
</ul>
<ul>
<li>霍桑东区（Hawthorn East）：中位价201.5万澳元，增幅34.2%</li>
</ul>
<ul>
<li>阳光区（Sunshine）：中位价64万澳元，增幅31.8%</li>
</ul>
<ul>
<li>艾文豪区（Ivanhoe）：中位价130.5万澳元，增幅26.7%</li>
</ul>
<ul>
<li>斯特拉斯莫区（Strathmore）：中位价110.05万澳元，增幅23.8%</li>
</ul>
<ul>
<li>克罗伊登区（Croydon）：中位价65.3万澳元，增幅23.2%</li>
</ul>
<ul>
<li>富士贵区（Footscray）：中位价73.025万澳元，增幅23.2%</li>
</ul>
<ul>
<li>吉斯巴拉区（Keysborough）：中位价63.1万澳元，增幅22.5%</li>
</ul>
<ul>
<li>本特利区（Bentleigh）：中位价132.9万澳元，增幅21.7%</li>
</ul>
<ul>
<li>切尔西区（Chelsea）：中位价73万澳元，增幅21.7%</li>
</ul>
<p>位于CBD东南的切尔西区，位于CBD东部远郊的克罗伊登区和位于CBD西部的阳光区在过去一年内都是<ahref="https://github.com/dahstt3965/djy/blob/master/gb/tag/%E6%88%BF%E4%BB%B7.md#1">房价</a>增长最快的区。<ahref="https://github.com/dahstt3965/djy/blob/master/gb/tag/%E6%88%BF%E5%9C%B0%E4%BA%A7.md#1">房地产</a>经纪公司Barry Plant位于吉斯巴拉的经纪人Ben Nguyen说，在房价高涨之前，居住在价格偏低的史滨威（Springvale）南区和诺贝公园区（Noble Park）的居民为升级住房，会在吉斯巴拉区购房。但现在吉斯巴拉区也吸引着克雷顿区（Clayton）以及一些滨海地带（包括阿斯本戴尔区（Aspendale）、埃迪思维尔区（Edithvale）和切尔西区）的居民到这里买房。</p>
<p>Nguyen说，他的公司在过去一年内转售的几栋住宅都增值超过10万澳元。一个典型案例是吉斯巴拉区一栋四居室住宅（45 Locharn Cr），该住宅在去年10月份以63.2万澳元售出，而该住宅在2014年6月时的买入价为481,498澳元。</p>
<p>吉斯巴拉区的房地产市场展现出两个不同的景象：一方面是位于老区的房地产，一方面是新开发的住宅区。Nguyen先生说：“新开发的住宅区过去是一片农场，住宅的增加使吉斯巴拉区的人口增加了一倍。”</p>
<p>在西部地区，富士贵区73.25万澳元的住宅中位价已接近大墨尔本地区的中位价。房地产经纪公司Biggin and Scott位于亚拉维尔区（Yarraville）的经纪人Tristan Tomasino说，富士贵区一直吸引着那些没有能力在亚拉维尔区、赛顿区（Seddon）和纽波特区（Newport）购买住宅的人们。</p>
<p>Tomasino说，富士贵区的住宅越来越高档化，原因是许多年轻家庭和职场人士从东部地区迁移至此，所以富士贵区的住宅价格越来越高。该区一栋优雅的四居室住宅（46 Stirling St）在今年3月份以134.6万澳元成功拍卖，而其在1998年的售价是28.5万澳元。在过去12个月内，富士贵区出现了许多宽敞的咖啡店和餐馆。Tomasino说，哪里的咖啡店高档，哪里的房产增值就快。</p>
<p>霍桑区和霍桑东区房价的增长幅度都超过了30%，为墨尔本房价增长最快的城区，中位价分别达到了211.65万澳元和201.5万澳元。都门集团首席经济学家Andrew Wilson认为，这俩个区的房价还可能会上涨。霍桑区有多种类型的房屋，在截至今年6月份的半年里，该区售出了一些高价住宅，也会对该区房屋的中位价产生影响。</p>
<p>本特利区的房屋中位价增长了21.7%，这里有一部分属于麦金农（McKinnon）中学学区。房地产经纪公司Buxton位于本特利区的经纪人Simon Pintado说，与滨海的布莱顿区（Brighton）、艾尔伍德区（Elwood）和桑均汉姆区（Sandringham）相比，本特利区和麦金农区的住宅价值较高。他说：“许多人为了购买合适的住宅，愿意在地理位置上做些让步。”Pintado认为，随着当地道路与铁路平交道口（Level Crossing）的拆除，本特利区的房价很可能还会上涨。</p>
<p>Wilson说，斯特拉斯莫区和艾文豪区的房价的高涨说明，有钱的买家仍然活跃在房地产市场。人们再次发现，CBD东北部位置适中的艾文豪区和海德堡区（Heidelberg）住宅有增值潜力，而且可能会延伸到格林斯伯劳区（Greensborough）、罗森纳区（Rosanna）和班杜拉区（Bundoora）。</p>
